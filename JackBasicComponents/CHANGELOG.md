# 更新日志

所有重要的变更都将记录在此文件中。

## [1.0.0] - 2025-11-16

### ✨ 新增功能

#### JackSlider - 滑块组件
- 支持自定义最小值、最大值和步长
- 支持自定义轨道颜色、选中颜色和滑块颜色
- 支持显示/隐藏步长刻度
- 支持显示/隐藏当前值提示
- 支持值变化回调函数
- 使用 `@Link` 装饰器实现双向数据绑定

#### JackCapsuleButton - 胶囊按钮组件
- 提供 6 种按钮类型：Primary、Secondary、Success、Warning、Danger、Info
- 提供 3 种按钮尺寸：Small、Medium、Large
- 支持朴素按钮样式（空心按钮）
- 支持禁用状态
- 支持加载状态（带加载动画）
- 支持自定义图标
- 支持完全自定义颜色（背景色、文字颜色、边框颜色）
- 智能处理点击事件（禁用和加载状态下自动阻止）

#### JackDatePicker - 日期选择器组件
- 支持 3 种选择模式：日期、时间、日期时间
- 支持显示农历
- 支持自定义日期范围（起始日期、结束日期）
- 支持自定义占位符
- 支持日期变化回调
- 提供美观的弹窗动画效果
- 使用 emoji 图标，无需额外资源文件

### 🎨 组件导出
- 提供统一的组件导出入口（Index.ets）
- 导出所有组件类、枚举类型和接口类型
- 提供便捷的 Builder 函数

### 📝 文档
- 完整的 README.md 文档
- 详细的 API 文档和使用示例
- 包含 DemoPage 演示页面

### 🔧 技术特性
- 基于 ArkTS 开发
- 遵循 HarmonyOS 组件开发规范
- 完整的 TypeScript 类型定义
- 避免了所有保留属性名冲突
- 使用空值合并运算符处理可选值

### 📦 打包配置
- 配置完善的 oh-package.json5
- 包含关键词和作者信息
- MIT 开源许可证

## 版本说明

### 语义化版本控制

本项目遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/) 规范：

- **主版本号**：当你做了不兼容的 API 修改
- **次版本号**：当你做了向下兼容的功能性新增
- **修订号**：当你做了向下兼容的问题修正

### 变更类型标识

- ✨ **新增功能** (Added)：新增的功能
- 🔄 **功能变更** (Changed)：现有功能的变更
- 🗑️ **废弃功能** (Deprecated)：即将移除的功能
- ❌ **移除功能** (Removed)：已移除的功能
- 🐛 **问题修复** (Fixed)：修复的 bug
- 🔒 **安全修复** (Security)：安全相关的修复
